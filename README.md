## Application Overview:
The Full Stack Task Manager is designed to provide a seamless experience for managing tasks efficiently. Users can register and log in to access a personalized dashboard where all their tasks are organized.

Upon logging in, the home page displays a summary of ongoing and upcoming tasks. Each task can have multiple subtasks, which can be added, edited, or marked as completed. The task pages provide an intuitive interface to track progress and manage priorities.

Adding a new task is simple and quick, with options to set details, deadlines, and subtasks. The application is fully responsive, ensuring that users can manage their tasks comfortably on desktops, tablets, or mobile devices.
